# CRUD Firebase 🔥

Este projeto foi desenvolvido com o intuito de ser um projeto para controle de estoque tendo como objetivo o registro de produtos automotivos dentro de uma estética automôtivo 🚗🧽.

## 💻 Tecnologias Utilizadas

- HTML 📱
- CSS 🎨
- JavaScript ☕
- Firebase 🔥
